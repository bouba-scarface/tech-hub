# JobsApi

All URIs are relative to https://api.salestim.io/v1.0

Method HTTP request Description
createProvisioningJob POST /jobs/provisioning Create a new provisioning job by sending a ProvisioningRequest
getJob GET /jobs/{id} Get detailed information about a job (Status, logs...)
